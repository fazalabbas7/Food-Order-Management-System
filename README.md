# 🍔 Food Order Management System

## 📌 Description
Food Order Management System is a backend-driven application designed to streamline restaurant operations by efficiently managing customer requests, table assignments, order processing, and billing workflows.

The system provides a structured flow from table request to order delivery and payment processing.

---

## 🚀 Features
- Table request & assignment system  
- Menu display and management  
- Order taking and tracking  
- Order preparation workflow  
- Order delivery handling  
- Bill generation  
- Payment processing  

---

## 🛠️ Tech Stack
- Java
